RNAseq Alignments - P.generosa Alignments and Alternative Transcript Identification Using Hisat2 and StringTie on Mox

2022
Miscellaneous
Published

September 14, 2022

As part of identifying long non-coding RNA (lncRNA) in Pacific geoduck(GitHub Issue), one of the first things that I wanted to do was to gather all of our geoduck RNAseq data and align it to our geoduck genome. In addition to the alignments, some of the examples I’ve been following have also utilized expression levels as one aspect of the lncRNA selection criteria, so I figured I’d get this info as well.

Trimmed RNAseq data from 20220908 was aligned to our Panopea generosa (Pacific geoduck) genome assembly, Panopea-generosa-v1.0.fa (FastA; 914MB), using HISAT2. Alternative transcripts and expression values were determined using StringTie. These were run on Mox./

Here’s a summary of the process:

  1. Generate necessary HISAT2 reference files (e.g. splic sites, exons, genome indexes).

  2. Concatenate FastQ files based on tissue/age and/or OA treatment.

  3. Run HISAT2 on concatenated FastQs.

  4. Run StringTie on grouped samples, with output formatted for import into Ballgown.

The SBATCH script below is very long. Skip to RESULTS section if you want.
